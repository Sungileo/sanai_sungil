천안시 아파트 매매가 분석

목적

  • 아파트 매매, 가격결정에 관한 유용한 인사이트 획득

데이터

활용 데이터는 2022-12월부터, 2023-01 까지의 천안시 아파트 매매내역 1059건이다.

아파트명, 전용면적, 거래금액, 건축년도, 층, 근처 역, 인구수, 크기, 외국인 비율, 초등학교 거리로 구성되어 있다.

전처리

  • 위도, 경도 데이터 제거

  • 컬럼명 영문 변환
